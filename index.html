Caesar Cipher

Teaching Material

The Caesar Cipher is a substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet.

To encrypt a message, each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet.

To decrypt a message, each letter in the ciphertext is replaced by a letter some fixed number of positions up the alphabet.

Monoalphabetic Cipher

Teaching Material

The Monoalphabetic Cipher is a substitution cipher where each letter in the plaintext is mapped to a unique letter in the ciphertext.

To encrypt a message, a key (substitution alphabet) is used to map each letter in the plaintext to a unique letter in the ciphertext.

To decrypt a message, the reverse mapping is applied using the same key.
